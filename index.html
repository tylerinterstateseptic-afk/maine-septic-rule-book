Help & Setup Guide

Getting an API Key

  1. On a computer, go to console.anthropic.com
  2. Create a free account or log in
  3. Click API Keys in the left menu
  4. Click Create Key, give it any name
  5. Copy the key — it starts with sk-ant-
  6. Open this app, tap Setup, paste the key
  7. Tap Load Rule Book and wait ~1-2 min

You only need to do this once. Everything is saved.

Add to Phone Home Screen

Android — Chrome
  1. Open this app in Chrome
  2. Tap the three-dot menu top right
  3. Tap Add to Home screen
  4. Tap Add
iPhone — Safari
  1. Open this app in Safari (not Chrome)
  2. Tap the Share button at the bottom
  3. Tap Add to Home Screen
  4. Tap Add
